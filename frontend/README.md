# Summary
- Project name: Forever - Ecommerce website
- Description: A modern frontend e-commerce website for the Forever brand, built with React and Vite.
# Description
- Forever - Ecommerce website is a personal frontend project built to practice React and Vite.  
- It simulates the customer side of an e-commerce website, allowing users to browse products, add them to the cart, and proceed with a mock checkout.  
# Installation
## Requirements
- Node.js >= 18
- Package manager: npm or yarn
## Steps
```bash
# 1. Clone the repository
git clone https://github.com/ninhh03/ecommerce_app
cd ecommerce_app

# 2. Install dependencies
npm install
# or
yarn install

# 3. Start the development server
npm run dev
# or
yarn dev

# 4. Build the project for production
npm run build

# 5. Preview the production build
npm run preview
```
# Features
- Browse, search and sort products
- Add and remove products from the cart
- Update product quantities in the cart  
- Show related products by category 
- Mock checkout flow
- Responsive design (mobile & desktop)
- User notifications with React Toastify
# Tech stack
- React 19
- Vite 7
- TailwindCSS 4
- React Router DOM 7
- React Toastify 11
